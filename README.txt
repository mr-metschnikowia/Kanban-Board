Kanban board CRUD web application built using vanilla JavaScipt (frontend) and Node.js (backend).

Instructions:
1. Install node and add to path.
2. Navigate to 'Kanban-Board' directory and run 'npm install' command to install all required node modules.
3. Run 'node P2main.js' command to run the web application.
4. Access the website by copying and pasting 'http://localhost3030' into Firefox web browser.
